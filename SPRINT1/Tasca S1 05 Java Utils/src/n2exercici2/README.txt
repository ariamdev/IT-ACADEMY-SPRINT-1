
-------------------------------------------------
EJECUTAR PROGRAMA JAVA DESDE LINEA DE COMANDOS:
-------------------------------------------------

1. Abrimos el Command Prompt (Windows):
- Podemos abrirlo buscandolo en el buscador de windows como "cmd" o
- Pulsando a la vez la tecla de Windows y la letra R. Se abrirá una pequeña ventana llamada "Ejecutar",
 escribe cmd y pulsa el botón Aceptar.

2.Indicar la ruta del archivo java:
- Escribiremos el comando "cd C:\ruta hasta el package del archivo". Por ejemplo: "cd C:\User\Desktop\IT Academy\src"
* cd : El comando cd sirve para poder ingresar a una carpeta existente dentro del directorio en donde nos encontremos.

2.Compilar el archivo java:
- Una vez estemos en el directorio, escribiremos el comando "javac nombrepackage/nombrearchivo.java".
Por ejemplo: "javac n1exercici1/Main.java"
*javac : El comando javac sirve para leer y compilar el archivo java a un archivo de clase.

3.Ejecutar el programa:
- Escribiremos el comando "java nombrepackage/nombrearchivo".
Por ejemplo: "java n1exercici1/Main n1exercici1". En la pantalla se nos ejecutará el programa java.

MENÚ DEL PROGRAMA:
*Opción 1: Listado de directory y su guardado de archivo:
-El archivo .TXT al seleccionar la opción 1 del menú del programa, se guardará en la misma ubicación desde
donde se ejecuta el programa Java.

*Opción 2: Leer archivo .txt
-Para leer el archivo .txt se debe indicar la ruta absoluta al archivo.

*Opción 3: Serializar objeto java a .ser
-Se pedirá un user name (objeto)
-Se pedirá la ruta donde guardar el archivo .ser . Se debe poner la ruta con una "\" final.
-Se pedirá el nombre del archivo .ser . Se debe añadir el .ser al final del nombre para crearlo correctamente.

*Opción 4: Leer archivo .ser
-Se pedirá la ruta absoluta absoluta donde está el archivo incluyendolo.
Ejemeplo: C:\Desktop\IT Academy\nombre.ser

**Si queremos volver atrás en la ruta del directorio debemos escribir el comando "cd..".
